# 简介

# hoshino_xcw

基于hoshino (opens new window),yobot (opens new window),go-cqhttp (opens new window)的开源Q群机器人

文档目前还在撰写中...

本项目源码hoshino_xcw (opens new window)

测试版可前往 Release (opens new window) 下载

本项目为开源项目，原则上是免费使用，可以联系作者获取源码或帮助搭建

如果你付费获取了本工具，可能是收取的服务器费用和托管费用或指导费用

bot维护不易，维护费用由维护组自定与本项目无关

关于在自己的服务器上搭建本bot，建议有能力者自行尝试

关于本Bot的功能与帮助，请前往帮助
