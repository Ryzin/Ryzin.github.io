# 概况

hoshino_xcw是公主连结的QQ群聊机器人,基于hoshino (opens new window),yobot (opens new window)实现.

# 自行需要搭建准备的东西有:

  1. 一台服务器，服务器推荐阿里或腾讯的最低配的云服务器（如果只用来搭bot，选用70-90RMB一年的1C2G1M的服务器）

    如果选择首次腾讯云还能通过QQ去找某人 (opens new window)实付打8折或者8.5折

  2. 基础的电脑知识

    复制粘贴/用编辑器修改代码/初中以上英语/会百度........

  3. 耐心

如果你看到这仍然有搭bot的信心就进行下一步吧
